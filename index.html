Langages maitrisés

Python

Python est un langage de programmation polyvalent, principalement connu pour sa simplicité et sa lisibilité. Il est généralement utilisé dans des domaines comme le développement web, l'analyse de données ou l'intelligence artificielle. Il possède de vastes bibliothèques qui facilitent un développement rapide et efficace, ce qui le rends utile pour de nombreuses implémentations.

Java

Java est un langage de programmation polyvalent qui est orienté objet. Il est utilisé dans le développement d'applications web, mobiles et d'entreprise. Sa robustesse, sa sécurité et son vaste écosystème de bibliothèques et frameworks permmettent de réaliser les projets complexes.

SQL

SQL est un langage de programmation conçu pour la gestion des bases de données. Il permet d'effectuer des opérations telles que la récupération, l'insertion la mise à jour et la suppression de données. De plus, grace à ce langage il est possible de gérer des bases de données. Ce langage est donc essentiel pour les projets avec serveur et base de données.

Bash

Bash est un interpréteur de commandes et un langage de script utilisé principalement sur les systèmes d'exploitation de type Linux. Bash offre une interface en ligne de commande permettant aux utilisateurs d'interagir avec le système d'exploitation en exécutant des commandes et de créer des scripts pour automatiser des tâches. Bash est un outil essentiel pour les systèmes Unix/Linux.

C# (sharp)

C# ("C sharp") est un langage de programmation moderne utilisé pour le développement d'applications Windows, web, mobiles et d'entreprise. Comme Java, une de ces caractéristiques principale est le programmation orientée objet. C# offre un environnement de développement robuste et productif très compatible avec les environnement Windows. Il est intéréssant pour ses performances, sa sécurité et son évolutivité.

Javascript

JavaScript ou JS est un langage de programmation utilisé pour le développement web. JavaScript est principalement utilisé pour ajouter des fonctionnalités interactives aux sites web,comme des animations, des formulaires dynamiques et des effets visuels. JavaScript est devenu un pilier du développement web. De plus, JavaScript est également utilisé pour le développement coté serveur, ce qui en fait un langage de programmation utilepour la création d'applications web modernes, animés et réactives.

HTML/CSS

HTML est le langage utilisé pour créer la structure et le contenu des pages web. En utilisant des balises prédéfinies telles que les développeurs peuvent organiser le contenu de manière sémantique et structurée. CSS est utilisé pour définir l'apparence et le style des sites web. Ensemble, ils représentent le nécéssaire du développement web moderne, et permet aux développeurs de créer des expériences web attrayantes et réactives qu'il est possible de combiner au JS.
